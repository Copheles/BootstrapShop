# BootStrap-Shop
Fully Featured Ecommerce Website with notifications system and advanced filtering functions.


### Development Stack

- React Front-End
- State management with Redux
- Redux Toolkit Query
- BootStrap
- Node js 
- Express Js
- MongoDB
- Socket Io
